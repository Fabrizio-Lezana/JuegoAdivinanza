# JuegoAdivinanza

## Descripción

Aplicación de consola desarrollada en Java que permite al usuario jugar a adivinar un número aleatorio generado por el sistema.

## Alcance del Proyecto

La aplicación se centra en un breve juego de adivinaza de un número aleatorio con un limite de intentos y simples pistas.

## Limitaciones del Proyecto
 - Las pistas son "mayor" o "menor".
 - No se le puede realizar otras consultas al juego.
 - No existe premio real por adivinar.
 - No contiene interfáz gráfica.

## Estructura del Proyecto

- Clase Main: permite interactuar con el usuario a través de un menú de opciones.
- Clase GestorJuego: contiene las funcionalidades principales para llevar a cabo el juego de adivinanza de un número aleatorio.
  
## Características implementadas

- Menú de opciones.
- Uso de estructuras de control:
  - if / else
  - switch
  - while
- Entrada y salida de datos con Scanner y System.out.println().
- Programación orientada a objetos mediante clases, atributos y métodos.
- Generación de números aleatorios.
- Validaciones de entrada de datos.
